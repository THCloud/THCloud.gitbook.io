# 121. 买卖股票的最佳时机

{% embed url="https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/" %}

```
// 动态规划，找最小的price以及当前和最小price的差值
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int mn = INT_MAX;
        int res = 0;
        for (auto& price : prices) {
            mn = min(mn, price);
            res = max(res, price - mn);
        }
        return res;
    }
};
```
