# 1480. 一维数组的动态和

{% embed url="https://leetcode.cn/problems/running-sum-of-1d-array/" %}

```
class Solution {
public:
    vector<int> runningSum(vector<int>& nums) {
        vector<int> res;
        int cur = 0;
        for (int i = 0; i < nums.size(); ++i) {
            cur += nums[i];
            res.push_back(cur);
        }
        return res;
    }
};
```
